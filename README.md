## ⚡ The one thing to know before you start

Git tracks **snapshots** of your project over time.
Think of it like save points in a video game — you can always go back.

GitHub is just a website that **hosts** those snapshots online so others can see them.

**Git ≠ GitHub.** Git is the tool. GitHub is a service that uses the tool.
